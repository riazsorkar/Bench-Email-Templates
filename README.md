# Bench Outreach Email Templates

A single-page HTML tool for **OutsourceToBD** that provides ready-to-use bench outreach email templates. Preview each template in the browser and copy the HTML body directly into Gmail or Outlook.

## Templates Included

| # | Template |
|---|----------|
| 1 | Registered Company — No Bench Post |
| 2 | Cold Outreach |
| 3 | Follow-Up |
| 4 | WhatsApp / DM |
| 5 | Personal Ask |
| 6 | Thanks for Posting |
| 7 | We'll Post for You |
| 8 | Referral Ask |

## Usage

1. Open `index.html` in any modern browser.
2. Click a tab to preview the desired template.
3. Click **Copy Email HTML** to copy the email body to your clipboard.
4. Paste directly into Gmail (compose → ⋮ → Paste as HTML) or Outlook.
